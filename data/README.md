# Data Directory

This folder contains the dataset used for the NBA Shot Selection Prediction project.

## Dataset Description

The dataset includes shot-level information such as:

- Shot distance  
- Shot location (court coordinates)  
- Shot type (2PT / 3PT)  
- Player information  
- Game context (quarter, time remaining, etc.)  

## Folder Structure

- raw/ → Original dataset (unprocessed data)  
- processed/ → Cleaned and feature-engineered dataset used for modeling  

## Note

Due to large file size limitations, the dataset is not included in this repository.
